# G5HubApi::Client

Gem to enable easy access to G5 Hub APIs

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'g5-hub-api-client'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install g5-hub-api-client
